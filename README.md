## About The Project

Trace is a program that allows all of the system calls used by a program to be retrieved. 

### Built With

* [C](https://devdocs.io/c/)

## Getting Started

### Prerequisites

Make

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:DamienMERCIER/Strace.git
   ```

## Usage

1. Compile 
   ```sh
   make
   ```
3. test it !
   ```sh
   ./strace strace
   ```
